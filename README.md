Gerenciador de Tarefas

Este projeto foi criado para facilitar a organização do dia a dia, permitindo registrar, visualizar e remover tarefas de forma simples. A ideia surgiu da necessidade de ter um sistema direto e prático para acompanhar rotinas pessoais ou de estudo.

Sobre o Projeto

A aplicação utiliza Node.js e Express no backend, garantindo uma estrutura leve e clara. Os dados são armazenados no MongoDB, permitindo que as informações fiquem salvas mesmo após o encerramento da aplicação. A arquitetura foi organizada para ser fácil de entender e manter.

Estrutura

O código foi dividido em pastas específicas, separando rotas, controladores e modelos para tornar o desenvolvimento mais fluido. O uso do .gitignore ajuda a manter o repositório limpo e livre de arquivos desnecessários.

Finalidade

Este repositório serve como base de estudo e referência para quem deseja aprender mais sobre APIs, organização de código e boas práticas no backend. Ele pode ser expandido e adaptado conforme as necessidades de cada usuário.
